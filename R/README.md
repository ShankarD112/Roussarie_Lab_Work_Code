# R analyses

This folder contains R scripts and helper functions for RNA-seq analysis and single-cell workflows.

## Subfolders

- `RNA_bulk/` — bulk RNA-seq analysis scripts (DESeq2, Shiny correlation viewer).
- `functions/` — reusable Seurat + single-cell helper functions.

## Usage notes

- Replace placeholder paths with your local paths.
- Some scripts are intended as templates; customize as needed.
